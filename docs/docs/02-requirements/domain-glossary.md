# ドメイン用語集（ユビキタス言語）

| 用語                              | 定義                                                                                                       |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| 休暇申請 (LeaveRequest)           | 社員が申請する休暇取得依頼                                                                                 |
| 申請者 (Applicant)                | 休暇申請を作成した社員                                                                                     |
| 判断者 (Judge)                    | 承認または却下を実行した社員（承認者を含む包括語）                                                         |
| 承認権者                          | 申請に対して承認 / 却下が可能な社員（同一組織かつ管理職階層に属する者）                                    |
| 承認権限                          | 申請に対して承認 / 却下を行える資格。所属組織コードと階層コードの組合せで判定する                          |
| 承認可否方針 (ApprovalPolicy)     | 申請者と承認者の所属・職層の関係から承認権限を判定するルール                                               |
| 部長層コード (Manager Layer Code) | 承認権限を持つと見なす職層 S コードの集合                                                                  |
| メイン所属                        | UserDto の `getMainUserPositionDto()` が返す主務所属                                                       |
| スナップショット                  | 申請時/判断時に取得した applicant/judge 情報を申請レコードに保存し、後の異動の影響を受けないようにする方式 |
| 申請中 (PENDING)                  | 申請受付後、承認 / 却下のいずれも未判断の状態                                                              |
| 承認済 (APPROVED)                 | 承認権者により承認された終端状態                                                                           |
| 却下済 (REJECTED)                 | 承認権者により却下された終端状態（却下コメントが必ず保全される）                                           |
| 業務エラー                        | 業務ルール違反として `MSTBusinessException` で送出され、画面に `FacesMessage` として提示される事象         |
