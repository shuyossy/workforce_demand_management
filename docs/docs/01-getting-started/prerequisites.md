# 前提環境

## 必須

- JDK 17 (Temurin) — アプリビルドおよび実行
- JDK 21 (Temurin) — VSCode Extension Pack for Java の動作要件
- WildFly 32.0.1.Final — APサーバ
- PostgreSQL 14+ — DB
- Node.js 22.x — フロントエンドビルド（Playwright / docusaurus 用）
- k6 — 性能テスト用 CLI

## 推奨

- VSCode + 推奨拡張（`.vscode/extensions.json` 参照）
- direnv — `.env` を自動読込（任意）
